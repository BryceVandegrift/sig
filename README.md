# sig

A rewrite of suckless [sic](http://tools.suckless.org/sic/) in Go

## About

sig is a rewrite of sic in Go.
It provides most of the same features, plus more.
The following improvements include:

- SSL/TLS support
- Easier to understand code base
- Concurrency
